### 🎮 Rock Paper Scissors Game ✊✋✌️

An interactive **Rock-Paper-Scissors game** created using **HTML, CSS, and JavaScript**. The player competes against the computer, with real-time results, score tracking, and the ability to reset the game.

✨ **Features**

* 🎨 Clean and responsive user interface
* 🖱️ Easy button-based gameplay
* 📊 Live score tracking for both player and computer
* 🔄 Reset option to start a new game
* ⚡ Smooth hover effects and transitions

🛠 **Tech Stack**

* 🧩 **HTML5** → Structure
* 🎨 **CSS3** → Styling & animations
* ⚙️ **JavaScript (ES6)** → Game logic and interactivity

🚀 Great project for practicing **DOM manipulation, event handling, and conditional logic** while building a fun game.
